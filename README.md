# Enterprise Traffic Capture and Network Analysis Lab

## Project Overview

This project demonstrates the implementation of enterprise network segmentation, SPAN port traffic mirroring, firewall-based routing, and network traffic analysis using Cisco Modeling Labs (CML), FortiGate Firewall, and Wireshark.

The objective was to validate VLAN segmentation, monitor inter-VLAN communication, and analyze network protocols using passive packet capture techniques.

---

## Objectives

* Design a segmented enterprise network
* Configure VLANs and inter-VLAN routing
* Implement SPAN traffic mirroring
* Capture and analyze network traffic
* Validate firewall policies and network segmentation
* Investigate network protocol behavior
